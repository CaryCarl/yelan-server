# 图片分组管理接口文档

### 基础信息

- **接口前缀**：`/admin/imagesGroup`
- **请求方式**：`POST`
- **Content-Type**：`application/json`

---

## 1. 分组列表查询

**接口地址**：`/admin/imagesGroup/getImageGroupList`

**接口说明**：从 `wallpaper_image_group` 表查询所有分组，支持分页、标题模糊查询、分类过滤、状态过滤、标签过滤和 is_webp 过滤

**请求参数**：

| 参数名 | 类型 | 必填 | 说明 | 示例 |
|--------|------|------|------|------|
| pageNumber | number | 否 | 页码，默认 1 | 1 |
| pageSize | number | 否 | 每页条数，默认 20 | 20 |
| title | string | 否 | 标题（模糊查询） | "风景" |
| categoryId | number | 否 | 分类ID | 24 |
| status | number | 否 | 状态：0隐藏 1显示 | 1 |
| tags | array | 否 | 标签ID数组（支持多选） | [1, 2, 3] |
| isWebp | number | 否 | 是否开启webp：0否 1是 | 1 |
| sort_field | string | 否 | 排序字段，默认 creation_time | "creation_time" |
| sort_order | string | 否 | 排序方向：ASC/DESC，默认 DESC | "DESC" |

**允许的排序字段**：

| 值 | 说明 |
|----|------|
| id | 分组ID |
| creation_time | 创建时间 |
| status | 状态 |
| is_webp | 是否开启webp |
| favorite_count | 收藏数 |
| group_images_total | 图片总数 |

**请求示例**：

```json
{
  "pageNumber": 1,
  "pageSize": 20,
  "title": "动漫",
  "categoryId": 24,
  "status": 1,
  "tags": [1, 2],
  "isWebp": 1,
  "sort_field": "creation_time",
  "sort_order": "DESC"
}
```

**响应参数**：

| 参数名 | 类型 | 说明 |
|--------|------|------|
| code | number | 状态码，200为成功 |
| msg | string | 提示信息 |
| data | array | 分组列表 |
| data[].id | number | 分组ID |
| data[].title | string | 标题 |
| data[].tagsId | string | 标签ID（逗号分隔） |
| data[].tagsIdArray | array | 标签ID数组 |
| data[].categoryId | number | 分类ID |
| data[].categoryName | string | 分类名称 |
| data[].creationTime | string | 创建时间 |
| data[].file | string | 文件夹路径 |
| data[].status | number | 状态 |
| data[].url | string | 封面URL |
| data[].favoriteCount | number | 收藏数 |
| data[].isWebp | number | 是否开启webp |
| data[].groupImagesTotal | number | 图片总数 |
| total | number | 总记录数 |
| pageNumber | number | 当前页码 |
| pageSize | number | 每页条数 |

**响应示例**：

```json
{
  "code": 200,
  "data": [
    {
      "id": 1311,
      "title": "11",
      "tagsId": "13",
      "tagsIdArray": [13],
      "categoryId": 24,
      "categoryName": "动漫",
      "creationTime": "2026-05-19T03:26:06.342Z",
      "file": "dajuzi/dajuzi/a-dongman/",
      "status": 1,
      "url": "https://xxx.cos.ap-guangzhou.myqcloud.com/...",
      "favoriteCount": 0,
      "isWebp": 1,
      "groupImagesTotal": 2
    }
  ],
  "total": 1,
  "pageNumber": 1,
  "pageSize": 20,
  "msg": "请求成功！"
}
```

## 2. 修改分组状态

**接口地址**：`/admin/imagesGroup/editImageGroupStatus`

**请求参数**：

| 参数名 | 类型 | 必填 | 说明 | 示例 |
|--------|------|------|------|------|
| id | number | **是** | 分组ID | 1 |
| status | number | **是** | 状态：0隐藏 1显示 | 1 |

**请求示例**：

```json
{
  "id": 1,
  "status": 1
}
```

**响应示例**：

```json
{
  "code": 200,
  "data": {
    "id": 1,
    "status": 1
  },
  "msg": "请求成功！"
}
```

## 3. 批量修改分组状态

**接口地址**：`/admin/imagesGroup/editImageGroupStatusBatch`

**请求参数**：

| 参数名 | 类型 | 必填 | 说明 | 示例 |
|--------|------|------|------|------|
| ids | array | **是** | 分组ID数组 | [1, 2, 3] |
| status | number | **是** | 状态：0隐藏 1显示 | 1 |

**请求示例**：

```json
{
  "ids": [1, 2, 3],
  "status": 1
}
```

**响应示例**：

```json
{
  "code": 200,
  "data": {
    "affectedRows": 3,
    "ids": [1, 2, 3]
  },
  "msg": "请求成功！"
}
```

## 6. 查询分组下的图片列表

**接口地址**：`/admin/imagesGroup/getImageListByGroupId`

**接口说明**：根据 groupId 查询 `wallpaper_images_list` 表中的图片

**请求参数**：

| 参数名 | 类型 | 必填 | 说明 | 示例 |
|--------|------|------|------|------|
| groupId | number | **是** | 分组ID | 1311 |
| pageNumber | number | 否 | 页码，默认 1 | 1 |
| pageSize | number | 否 | 每页条数，默认 20 | 20 |
| status | number | 否 | 图片状态：0隐藏 1显示 | 1 |
| isWebp | number | 否 | 是否开启webp：0否 1是 | 1 |

**请求示例**：

```json
{
  "groupId": 1311,
  "pageNumber": 1,
  "pageSize": 20,
  "status": 1,
  "isWebp": 1
}
```

**响应参数**：

| 参数名 | 类型 | 说明 |
|--------|------|------|
| code | number | 状态码，200为成功 |
| msg | string | 提示信息 |
| data | array | 图片列表 |
| data[].id | number | 图片ID |
| data[].groupId | number | 分组ID |
| data[].tagId | string | 标签ID（逗号分隔） |
| data[].tagIdArray | array | 标签ID数组 |
| data[].imageUrl | string | 图片URL |
| data[].status | number | 状态 |
| data[].likeCount | number | 喜欢数 |
| data[].favoriteCount | number | 收藏数 |
| data[].viewCount | number | 浏览量 |
| data[].isWebp | number | 是否开启webp |
| data[].createTime | string | 创建时间 |
| data[].updateTime | string | 更新时间 |
| total | number | 总记录数 |
| pageNumber | number | 当前页码 |
| pageSize | number | 每页条数 |

**响应示例**：

```json
{
  "code": 200,
  "data": [
    {
      "id": 15,
      "groupId": 1311,
      "tagId": "13",
      "tagIdArray": [13],
      "imageUrl": "https://xxx.cos.ap-guangzhou.myqcloud.com/...",
      "status": 1,
      "likeCount": 0,
      "favoriteCount": 0,
      "viewCount": 0,
      "isWebp": 1,
      "createTime": "2026-05-19 11:26:06",
      "updateTime": "2026-05-22 02:23:18"
    }
  ],
  "total": 2,
  "pageNumber": 1,
  "pageSize": 20,
  "msg": "请求成功！"
}
```
